Environment

Environment

The tool belt.

  • Node
  • Visual Studio Code
  • Google Chrome

Node

Install the latest LTS version of Node. Version 8.x of Node will be used for this book. 

$ node -v
# 8.x

Node comes with node package manager (npm).

npm -v
# 5.x

Visual Studio Code

Visual Studio Code. It's my code editor of choice. It's awesome. Check it out (or don't, any code editor suited for web development will work).

Google Chrome

Google Chrome, with React Developer Tools installed, will be used throughout this book for viewing and troubleshooting React code.
